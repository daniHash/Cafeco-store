# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/daniHash/Cafeco-store.git
```

Go to the project

```bash
cd cafeco-store
```

Install dependencies

```bash
npm install
```

Start JSON Server

```bash
npm run server
```

Start the development server

```bash
npm run dev
```

---

# 📌 Environment

```
Node.js >= 18
npm >= 10
```
